# Next.js Project & Demo Workspace

A personal workspace for learning, experimenting, and practicing Next.js development. This repository contains various demo projects and files created during my Next.js learning journey.

## 📌 Purpose

This repository serves as my:
- **Learning playground** for Next.js concepts
- **Demo workspace** for testing new features
- **Project hub** for office-related Next.js development
- **Reference library** for reusable code patterns

## 🗂️ Project Structure
```bash
├── app/
│   ├── layout.tsx          # Root layout with navigation
│   ├── page.tsx            # Homepage
│   ├── globals.css         # Global styles
│   └── about/
│       └── page.tsx        # About page
├── public/                 # Static assets
├── node_modules/           # Dependencies
├── package.json            # Project metadata
├── tsconfig.json           # TypeScript configuration
├── next.config.ts          # Next.js configuration
└── tailwind.config.js      # Tailwind CSS configuration
└── README.md               # This file
```

> **Note:** This is a dynamic workspace. New demo folders are added regularly as I learn and practice.

## 🚀 Technologies Used

- **Next.js** (v14+ / v16+)
- **React**
- **TypeScript** / JavaScript
- **Tailwind CSS**
- **Turbopack**

## 🛠️ Common Commands

Navigate to any demo folder and run:

```bash
cd demo-folder-name
npm install          # Install dependencies
npm run dev          # Start development server
npm run build        # Create production build
```

> **📝Note:** This is a personal learning repository - code may not be production-ready. Demo folders are independent with different configurations.
